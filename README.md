# Longitudinal arm swing
A repository containing code related to the publication: <b>'Progression of digital arm swing measures during free-living gait in early Parkinson disease'</b>.

The repository contains code to transform weekly aggregates of arm swing into results shown in the article. The notebooks should be run chronologically, and require data that will later be shared publicly alongside setting the repository to public.

Note: the repository is currently set to private and shall be set to public once the article is sent to the editor.
